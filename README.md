# canvas-drawing — SVG 다이어그램 편집기

순수 바닐라 JavaScript와 SVG로 구현한 웹 기반 다이어그램 · 도형 편집 도구입니다.

## 주요 기능

- **도형 추가** — 패널에서 도형을 드래그하여 캔버스에 배치
- **도형 편집** — 이동, 선택, 삭제, 라벨 변경 (더블클릭)
- **연결선** — 도형 간 베지어 곡선으로 연결
- **자유 그리기** — 펜 도구로 자유형 경로 그리기
- **지우개** — 자유 그리기 경로 삭제
- **스타일 조정** — 색상 및 선 굵기 설정

## 기술 스택

| 분류 | 기술 |
|------|------|
| 마크업 | HTML5 |
| 렌더링 | SVG |
| 로직 | Vanilla JavaScript (ES6+) |
| 스타일 | CSS3 |

외부 라이브러리 없이 순수 웹 기술만으로 구현했습니다.

## 실행 방법

별도 설치 없이 `index.html`을 브라우저에서 열면 바로 실행됩니다.

```bash
git clone https://github.com/tjrqh/311.git
cd 311
open index.html
```
